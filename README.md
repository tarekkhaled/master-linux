### Setup Front-end
```
cd client/
npm install 
npm start
```


### Setup Back-end
```
cd server/
npm install
create .env file with (DATABASE=mongodb connection, JWT_SECRET)
npm start || npm run dev (with logging)
```


### Setup Full-stack
```
npm install
npm run full-stack
```
